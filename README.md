# Mathracer

Competitive math game to practice your mental math in a fun and exciting way!!

### Tech stack:
Frontend:
- NextJs
- TailwindCss
- Deployed on Google Cloud Run

Backend:
- C#
- ASP.NET Api, SignalR
- Deployed on Google Cloud Run
